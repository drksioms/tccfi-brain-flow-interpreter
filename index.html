TCCFI/TCCS/TCD Brain Flow Interpreter

医療者向けの補助解釈ツールです。診断や治療判断を代替せず、神経診察、画像、 ICP/CPP、EEG、臨床経過などとの統合を考慮してください。

PI
--
PI / 拍動指数 (PSV - EDV) / MFV。高値は末梢抵抗上昇や低CPP patternの可能性を示唆しますが、ICP推定には使用しません。
RI
--
RI / 抵抗指数 (PSV - EDV) / PSV。抵抗性の参考指標です。単独で診断や治療判断には使用しません。
Lindegaard
--
Lindegaard Ratio / MCA-ICA比 MCA MFV / 同側頭蓋外ICA MFV。SAHでvasospasm physiologyの補助評価に使います。ICA未入力時はprovisionalです。
BA/VA
--
BA/VA Ratio / BA-VA比 BA MFV / 頭蓋外VA MFV。後方循環vasospasm physiologyの補助評価に使います。VA未入力時はprovisionalです。
